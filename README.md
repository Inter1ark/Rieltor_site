# Правильная Локация

Премиальный одностраничный сайт независимой инвестиционной экспертизы недвижимости и бизнеса. Тёмная «дорогая» эстетика, кинематографичный hero на скролл-видео и продуманная статичная часть.

## Ключевые особенности

- **Scroll-controlled hero** — фон проигрывается покадрово через секвенцию AVIF на `<canvas>` (надёжно на iOS Safari). Пошаговый режим: на каждом скролле видео плавно доматывается до нужного кадра, между кадрами cross-dissolve для эффекта 60 fps.
- **Горизонтальная галерея** в стиле phantom.moscow: pinned-секция, при вертикальном скролле ряд панелей едет по горизонтали.
- **Плавная смена фона** при скролле, full-bleed картиночные секции, бенто-раскладки, micro-interactions.
- **Плавный инерционный скролл** на Lenis, анимации появления по скроллу.
- **Разделы**: подход, услуги, цена ошибки, процесс, кейсы, блог, FAQ, форма заявки.
- **Роуты**: `/` (лендинг), `/cases` + `/cases/[slug]`, `/blog` + `/blog/[slug]` (SEO-статьи).
- **Форма заявки** с отправкой в Telegram (`/api/lead`).
- SEO: метаданные, Open Graph, статическая генерация страниц.

## Технологии

- Next.js 16 (App Router) + React 19 + TypeScript
- Tailwind CSS v4
- Framer Motion (`motion`) + Lenis
- next/image, шрифты Manrope + Geologica

## Запуск

```bash
npm install
npm run dev
```

Открыть http://localhost:3000

### Переменные окружения (необязательно)

Для отправки заявок в Telegram создайте `.env.local`:

```
TELEGRAM_BOT_TOKEN=...
TELEGRAM_CHAT_ID=...
```

Без них форма работает, заявка пишется в консоль сервера.

## Кадры hero

Кадры лежат в `public/frames/{desktop,mobile}` (AVIF). Генерируются из исходного видео через ffmpeg (исходник в репозиторий не входит).
